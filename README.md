# Basic CRUD with Angular 14 and NodeJS with Express

Basic CRUD example on Angular 14 and NodeJS with Express. Use as base for more advanced examples.

## Angular Client

### Install

    npm install

### Run

    npm start


## Node Server

### Install

In a separate terminal, run:

    cd node-server
    npm install

### Run

    npm run start

or with nodemon:

    npm run indev
